1) install Grafana is simple, just go to the web site.

2) after install, Grafana server is needed to be started (better make it auto start when PC started).

4) the database we use is call influxDB. install can be found in there web site.

5) user manual can be found in influxDB, look at the "Documents" section.

6) the language of influxDB is same as SQL.

7) the influxDB server is also needed to be started.

8) the databases are stored in /var/lib/influxdb/data/
